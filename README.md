# Lightspeed is made from TrebleShot libre software for shaing Files in Android
Send and receive files over available connections. [FAQ](https://github.com/genonbeta/TrebleShot/blob/master/FAQ.md).

## Build from source
Required tools;
* Build tools: 28.0.3
* Support library: 1.0.0 - androidx
* Gradle plugin: 3.2.1
* Android Studio: 3.2.1
